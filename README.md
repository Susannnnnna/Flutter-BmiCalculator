## How to run the app?
### Environment
- Windows 11 Pro v22H2

### Used technologies
- Android Studio (device: Nexus 6 API 34)
- Flutter
- Dart

### Run the app
Download files from github
```
git clone https://github.com/Susannnnnna/Flutter_BmiCalculator
```
Open application in Android Studio
